# YOLO标注工具 (YOLO Label Creator)

## 项目概述

YOLO标注工具是一个用于创建和管理YOLO格式目标检测数据集标注的桌面应用程序。该工具提供了直观的图形界面，支持边界框绘制、类别管理、自动标注等功能，帮助用户高效地为图像数据集创建标注。

## 主要功能

- **图像浏览与管理**：支持浏览和管理项目目录中的图像文件
- **边界框绘制**：直观的绘制、调整和删除边界框
- **类别管理**：添加、编辑和管理目标类别
- **YOLO格式支持**：自动转换为YOLO格式的标注文件
- **自动标注**：集成YOLOv8模型，支持自动标注功能
- **快捷键支持**：可自定义的快捷键，提高标注效率
- **多语言支持**：内置中文界面
